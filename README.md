Shopping Cart Application
Overview

This is a responsive Shopping Cart Application built using HTML, CSS, and JavaScript. The application allows users to browse food products, add items to a shopping cart, adjust quantities, remove products, and confirm orders through a checkout modal.

The project was developed to strengthen practical frontend development skills, particularly in DOM manipulation, state management, event handling, responsive design, and browser storage.

Features
Product Management
Display products dynamically from JavaScript data.
Product cards with images, category, name, and price.
Responsive product grid layout.
Shopping Cart
Add products to cart.
Increase product quantity.
Decrease product quantity.
Remove products from cart.
Automatic cart item counting.
Automatic total price calculation.
Order Management
Order confirmation modal.
Order summary before checkout.
Start a new order functionality.
Clear cart after order completion.
Data Persistence
Cart data stored in Local Storage.
Cart remains available after page refresh.
User Interface
Responsive design for mobile and desktop devices.
Interactive buttons and hover effects.
Modern card-based product layout.
Sticky cart panel on larger screens.
Technologies Used
HTML5
CSS3
JavaScript (ES6)
Local Storage API
